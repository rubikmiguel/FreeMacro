
# FreeMacro - Open Source Peripheral Automation

FreeMacro is a high-precision macro engine developed in Java 8. It provides low-level control over mouse and keyboard events, allowing for complex automation tasks with full transparency and freedom.

## Key Features
* **Granular Mouse Control**: Separate Press and Release events for advanced interactions.
* **Low-Level Simulation**: Automate drag-and-drop and precise movements that standard tools can't handle.
* **Extensible Architecture**: Designed to be easily modified for specific automation needs.
* **Modern UI**: Powered by the FlatLaf Look and Feel.

## Technical Requirements
* **Java Runtime**: Version 8 (1.8) or higher.
* **Dependencies**: FlatLaf (included in the `/libs` folder).

## Setup & Contribution
1. Clone the repository.
2. Add the `.jar` files from `/libs` to your Classpath.
3. Run `com.app.Main`.
